# pandoc-minted

It is pandoc filter script.

It use [minted](https://github.com/gpoore/minted) alternative listings for CodeBlock.

# How to use

Build by cabal.

~~~
pandoc --filter=pandoc-minted
~~~

```md
\newcommand{\mintlang}{hs}
```
